Latin Extended Additional

U+1E00 – U+1EFF   (7680–7935)

Combining Diacritical
Marks Supplement		   Greek Extended

These characters are used with Basic Latin. Characters 7808-7813, 7922 and 7923 are present in Microsoft’s WGL4 character set. Some of the characters in this range are used in Vietnamese; you can see some Unicode Vietnamese on the Nguoi Viet Online site, and you can see some non-Unicode Vietnamese on the VASC Orient Web site. You can find information about Unicode for the Vietnamese Language on the Vovisoft Web site.

Unicode Vietnamese is supported in all versions of Microsoft Windows from Windows 95 onwards, although older versions need to have Vietnamese Language Support installed from the Windows Update site. Mac OS X 10 supports Unicode Vietnamese, but Mac OS 9 does not (none of the Language Kits include the Latin Extended Additional range).

The characters that appear in the first column of the following table depend on the browser that you are using, the fonts installed on your computer, and the browser options you have chosen that determine the fonts used to display particular character sets, encodings or languages.
